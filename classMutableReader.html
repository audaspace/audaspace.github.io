Detailed Description

This class represents a reader for a sound that can change with each playback.

The change will occur when trying to seek backwards If the sound doesn't support that, it will be restarted.

Warning
Notice that if a SoundList object is assigned to several MutableReaders, sequential playback won't work correctly. To prevent this the SoundList must be copied.
